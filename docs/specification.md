# CsStructureViewer 仕様書

## 1. 目的・背景

バイブコーディング（AI支援によるコーディング）の過程で、AIによる実装の設計を可視化・チェックするためのツール。  
C# および Unity プロジェクトにおけるクラス構造と依存関係を視覚的に把握することを目的とする。

---

## 2. 対象プロジェクト

- C# プロジェクト（.NET）
- Unity プロジェクト（C# スクリプト）

---

## 3. 表示仕様

### 3.1 クラス矩形

- クラスを矩形で表現する
- 矩形内にクラス名を表示する
- 矩形の塗りつぶし色は、所属する名前空間矩形と同系色とする

### 3.2 依存関係の表示

- クラス間の依存関係をクラス矩形を結ぶ線で表現する
- 線のスタイルは UML の依存関係矢印に準拠する（矢印付きの線）
- ※依存関係の種類（継承・実装・参照など）の区別については今後の仕様追加で定義する

### 3.3 名前空間矩形

- クラスが属する名前空間をグループ矩形として表示する
- 名前空間はフォルダ名を基準として決定する
- 矩形の枠上に名前空間名を表示する（例: `-- NamespaceName ------`）
- 矩形は半透明の塗りつぶしで表示する
- 名前空間ごとに異なる色を割り当てる
- 名前空間矩形の中に、その名前空間に属するクラス矩形を配置する
